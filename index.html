Exemplos de Funções:

  • Seno (sen): Representa a coordenada Y no círculo. Período: 2π. Imagem: [-1, 1].
  • Cosseno (cos): Representa a coordenada X no círculo. Período: 2π. Imagem: [-1, 1].
  • Tangente (tan): Razão `sen(x) / cos(x)`. Período: π. Imagem: (-∞, +∞).
  • Seno (2x): Frequência dobrada. Período: π. Imagem: [-1, 1].
  • 5 + sen(3x-2): Função deslocada e comprimida.
    • - Período: 2π/3 (frequência 3x)
    • - Imagem: [4, 6] (Amplitude 1, deslocada 5 para cima)
    • - Fase: Deslocada `2/3` para a direita.
